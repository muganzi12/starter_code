This project is a RESTful e-commerce API built using Django and Django REST Framework.
It provides endpoints for managing products and customer orders efficiently.
The API supports listing, creating, and retrieving products based on stock availability.
Order management functionality includes creation, listing, and detailed retrieval.
Optimized database queries are implemented using prefetch_related for performance.
The system also provides basic analytics such as total product count and maximum price.
It follows clean architecture principles using Django generic class-based views.
The API is designed to be easily extensible and integration-ready.
It can serve as a backend for web, mobile, or third-party systems.
Overall, it demonstrates best practices for building scalable REST APIs in Django.
